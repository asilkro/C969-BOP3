Last updated: 7/9/2022

Notes for Assessor:
1) The log file that documents logon attempts should be generated and updated in the directory this application
launches from (System.Environment.CurrentDirectory).

2) Supported locales/languages are US-English & Germany-German.

3) Business hours are defined as seven days a week from 0900 to 2100 local time.

4) Appointment reminder alerts are only generated on logon.
